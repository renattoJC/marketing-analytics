# Marketing Campaign Analysis

## Project Description
This project is focused on analyzing the effectiveness of various marketing campaigns to determine their impact on sales and customer engagement. The aim is to provide insights into which campaigns yielded the best results and to optimize future marketing strategies.

## Features
- Interactive data visualizations to illustrate campaign performance.
- Comparison of different campaigns based on various metrics such as ROI, conversion rates, and customer acquisition cost.
- Detailed reports on campaign analytics for better decision-making.

## Requirements
- Python 3.6 or higher
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/renattoJC/Proyectos-Variados-en-Python-y-Automatizaci-n.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Proyectos-Variados-en-Python-y-Automatizaci-n
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- To run the main analysis script, execute:
  ```bash
  python analyze_campaign.py
  ```
- For detailed visualizations, access the Jupyter notebooks and run the cells.

## Project Structure
- `data/`: Directory containing the datasets used for analysis.
- `notebooks/`: Jupyter notebooks for exploratory data analysis.
- `scripts/`: Python scripts for data processing and analysis.
- `requirements.txt`: A file listing all required packages.

## Data Analysis Methods
- Descriptive statistics to summarize basic features of the data.
- A/B testing to compare the performance of different campaigns.
- Regression analysis to understand the relationship between marketing spend and sales.

## Key Insights
- The analysis revealed that Campaign A had a significantly higher conversion rate compared to Campaign B, suggesting better targeting and messaging.
- Seasonal trends were identified, indicating the best times to run specific campaigns for maximum engagement.
